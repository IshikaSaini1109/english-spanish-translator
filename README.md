# English to Spanish Translator

This project is a Machine Learning based English to Spanish Translator built using Natural Language Processing techniques.

## 📌 Project Description
The model translates English sentences into Spanish using a sequence-to-sequence encoder-decoder architecture.

It demonstrates how neural networks can be used for machine translation tasks.

## 🚀 Features
- Translate English text into Spanish
- Uses NLP preprocessing techniques
- Built using deep learning models

## 🛠 Technologies Used
- Python
- TensorFlow / Keras
- Natural Language Processing (NLP)

## 📂 Project Structure
dataset/ – translation dataset  
model/ – trained model files  
notebook.ipynb – training notebook

## ▶️ How to Run the Project
1. Clone the repository
2. Install required libraries
3. Run the notebook or Python script

## 📷 Output Example
Input: "Hello, how are you?"  
Output: "Hola, ¿cómo estás?"
